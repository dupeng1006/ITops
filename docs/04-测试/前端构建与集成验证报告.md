## 8. 问题与处理

| 问题 | 处理 |
|---|---|
| SPA 回退初版用 `fastapi.HTTPException` 捕获，StaticFiles 实际抛 `starlette.exceptions.HTTPException`（父类），`/recon/m1` 未回退 | 改捕 starlette 异常 |
| `/api/nonexistent` 被回退吃掉：Starlette `get_path` 按 OS 归一化，Windows 下路径分隔符为反斜杠（`api\nonexistent`） | 判断前统一 `replace("\\","/")` 归一化，已复验七项行为 |
| npm 在 Git Bash 不可直呼（npm.cmd） | 全程绝对路径 + `npm.cmd` 调用，build.bat 中 `set PATH=%NODE_HOME%;%PATH%` 后 `call npm.cmd` |
