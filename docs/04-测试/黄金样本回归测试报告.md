## 1. 样本场景覆盖表

| 场景 | 样本设计 | 预期行为 | 实际结果 |
|------|----------|----------|----------|
| a 精确匹配差异≤1% | ZH001/ZH011~ZH015 等 6 组 | 精确匹配、无色 | ✅ 一致 |
| b 精确匹配差异>1% | ZH002（净值差 2.0725%） | 浅红 FFCCCC、计入差异统计 | ✅ 一致 |
| c 映射 | 净值表 7001 → 6001 | 1 条映射、与基金表 6001 精确匹配 | ✅ 一致 |
| d 映射去重冲突 | 原始 6001（5,000 万）+ 映射 6001（8,000 万） | 删原始留映射，差异 0.375% 无色（若失效差异 60.6% 标红，金丝雀） | ✅ 一致 |
| e I9 过滤 | C1004 一含 I9 一不含；C1005 两条全含 I9 | 前者删 I9 留非 I9；后者全保留、精确匹配 1 对 2 复制 | ✅ 一致 |
| f 大宗产品 | ZH006(AZ0206)、ZH007(AZ0205) | 标橙 FFC000、备注"大宗产品无需核对"、不计入差异统计 | ✅ 一致 |
| g 模糊匹配 | "测试稳益11号" vs "测试稳益11号A类" | 相似度 0.875 ≥ 0.5，模糊匹配；产品名称列取净值表名，代码列保持 C1008 | ✅ 一致 |
| h 未匹配 | ZH009 无任何对应 | 浅黄 FFFF99、置底、查询侧为空 | ✅ 一致 |
| i 代码格式陷阱 | 净值表 6101 为数值型、基金表为字符串 | astype(str) 后精确匹配 | ✅ 一致 |

  黄金样本回归基线建立完成，可作为后续迭代（规则库迁移、FastAPI 接入、
  依赖升级）的防回归门禁；

- 注意：本基线锁定 pandas 3.0.2 / openpyxl 3.1.5 行为，二期安装
  `requirements.txt` 依赖后应重跑回归确认第三方库版本未引入行为漂移。
